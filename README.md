# MxHub Exchange Duo

Welcome to MxHub Exchange Duo, a comprehensive platform for managing motocross racing data, user accounts, and digital currency exchange.

This application provides a unified interface for:
-   **Dashboard Overview**: A central hub to view key statistics and upcoming events.
-   **Race Management**: Track and update race data.
-   **Digital Wallet**: Manage Gold Coins and Sweeps Coins.
-   **Exchange Services**: Transfer Sweeps Coins for redemption.
-   **Admin Panel**: A backend interface for data input and content management, including an AI-powered summarization tool.

## Getting Started

Follow these instructions to get a local copy up and running.

### Prerequisites

-   Node.js (v18 or later)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```sh
    git clone <your-repo-url>
    ```
2.  Navigate to the project directory:
    ```sh
    cd mxhub-exchange-duo
    ```
3.  Install NPM packages:
    ```sh
    npm install
    ```

### Running the Development Server

To start the development server, run:

```sh
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

You can start editing the main page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
